Que es el DOM

Document Object Model o DOM nos permite acceder y manipular las páginas HTML. Se diseñó originalmente para manipular de forma sencilla los documentos XML.

Se trata de un estándar de W3C, es una interfaz de programación para los documentos HTML y XML que representa el documento de forma estructurada (de árbol) y delimita de qué forma los programas pueden acceder para modificar su estructura, estilo y contenido. El DOM da una representación del documento como un grupo de nodos y objetos que tienen propiedades y métodos.

El DOM es una representación completamente orientada al objeto de la página web y puede ser modificado con un lenguaje de script como JavaScript.

Árbol de Nodos

Con JavaScript podemos manipular las páginas web accediendo a los “nodos” del DOM para transformarlos de forma dinámica.

Los navegadores transforman el documento html de forma automática y nos permiten utilizar las herramientas de DOM. El DOM transforma todos los documentos HTML en un conjunto de elementos llamados nodos, que están interconectados y que representan los contenidos de las páginas web y las relaciones entre ellos. Se describe como "árbol de nodos" debido a su estructura.

Algunos de los tipos de nodos mas importantes son

  • Document: nodo raíz del que derivan todos los demás nodos del árbol.
  • Element: representa cada una de las etiquetas HTML. Se trata del único nodo que puede contener atributos y el único del que pueden derivar otros nodos.
  • Attribute: se define un nodo de este tipo para representar cada uno de los atributos de las etiquetas HTML, es decir, uno por cada par atributo=valor.
  • Text: nodo que contiene el texto encerrado por una etiqueta HTML.
  • Comment: representa los comentarios incluidos en la página HTML.

Metodos y propiedades más utilizados

Métodos:

  • document.getElementById(id): Encuentra un elemento por su id.
  • document.getElementsByTagName(name): Encuentra un elemento por su tag name.
  • document.getElementsByClassName(name): Encuentra un elemento por su class name.
  • document.createElement(element): Crea un elemento
  • document.removeChild(element): Remueve un elemento.
  • document.appendChild(element): Añade un elemento
  • document.replaceChild(new, old): Remplaza un elemento
  • document.write(text): Escribe dentro del domumento HTML
  • element.addEventListener(type, listener): Añade a un elemento para que realice una función al interactuar con el.

Propiedades:

  • element.innerHTML = new html content: Cambia el HTML dentro de un elemento
  • element.attribute = new value: Cambia el valor del atributo de un elemento
  • element.style.property = new style: Cambia el estilo de un elemento

Eventos

Los eventos de JavaScript permiten la interacción de los usuarios con una web a través de JavaScript. La pulsación de una tecla, hacer click o la carga una página se consideran eventos.

En la versión 4 del estándar HTML se introdujo el modelo simple de eventos que se considera parte del nivel más básico de DOM. Sus características son limitadas, pero es el único modelo que es compatible con todos los navegadores y por lo tanto, el único que permite crear aplicaciones que funcionan de la misma manera en todos los navegadores.

TIPOS DE EVENTOS

El nombre de los eventos se construye mediante el prefijo on, seguido del nombre de la acción asociada al evento. Por ejemplo : “onclick”.

Eventos más importantes definidos por JavaScript:

onblur Un elemento pierde el foco <button>, <input>, <label>, <select>, <textarea>, <body>
onchange Un elemento se modifica <input>, <select>, <textarea>
onclick Pulsar y soltar el ratón Todos los elementos
ondblclick Pulsar dos veces seguidas con el ratón Todos los elementos
onfocus Un elemento obtiene el foco <button>, <input>, <label>, <select>, <textarea>, <body>
onkeydown Pulsar una tecla y no soltarla Elementos de formulario y <body>
onkeypress OBSOLETO Pulsar una tecla Elementos de formulario y <body>
onkeyup Soltar una tecla pulsada Elementos de formulario y <body>
onload Página cargada completamente <body>
onmousedown Pulsar un botón del ratón y no soltarlo Todos los elementos
onmousemove Mover el ratón Todos los elementos
onmouseout El ratón "sale" del elemento Todos los elementos
onmouseover El ratón "entra" en el elemento Todos los elementos
onmouseup Soltar el botón del ratón Todos los elementos
onreset Inicializar el formulario <form>
onresize Modificar el tamaño de la ventana <body>
onselect Seleccionar un texto <input>, <textarea>
onsubmit Enviar el formulario <form>
onunload Se abandona la página, por ejemplo al cerrar el navegador <body>

Los eventos se utilizan a través de los manejadores de eventos, que pueden ser incluidos como atributo de un elemento HTML o los Manejadores semánticos asignados mediante DOM sin necesidad de modificar el código HTML de la página.

La diferencia entre estas dos formas que de la segunda (utilizando element.addEventListener(click, miFuncion)) se pueden añadir multiples eventos a un solo elemento, mientras que si lo hacemos adjuntandolo al atributo de un elemento sólo podemos usar uno ( onclick=”miFuncion()”).

function muestraOculta(id) {
    var elemento = document.getElementById('contenidos_' + id);
    var enlace = document.getElementById('enlace_' + id);

else {
    enlace.textContent = 'Ocultar contenidos';
}
}

document.onmousemove = posicionRaton
function posicionRaton(event) {
    event = event;
    coordenadaX = event.clientX;
    coordenadaY = event.clientY;
    muestraInformacion(['Ratón', 'Posición: X: ' + coordenadaX + ' Y: ' +     coordenadaY]);
}

document.onkeydown = infoTecla;
function infoTecla(event) {
    var event = event;
    var caracter = event.charCode || event.keyCode;
    var letra = String.fromCharCode(caracter);
    var codigo = letra.charCodeAt(0);
    muestraInformacion2(['Teclado', 'Carácter [' + letra + ']', 'Código [' +     codigo + ']']);
}

function muestraInformacion(mensaje) {
    document.getElementById("info").innerHTML = '<h1>' + mensaje[0] + '</h1>';

    }
}
