Qué es JavaScript

JavaScript o JS es un lenguaje de programación interpretado, es decir, es un lenguaje en el que el código fuente se ejecuta directamente, instrucción a instrucción, el código no pasa por un proceso de compilación, sino que tenemos un programa llamado intérprete que lee la instrucción en tiempo real, y la ejecuta.

JS es más conocido como un lenguaje para páginas web pero como es mencionado en MDN Web Docs, es usado también en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat. JavaScript es un lenguaje de programación basada en prototipos, en palabras simples: permite la creación de un objeto sin definir primero su clase. También es multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa (por ejemplo programación funcional).

No se debe confundir JavaScript con el lenguaje de programación Java. Ambos "Java" y "JavaScript" son marcas de Oracle en los Estados Unidos y otros países, Sin embargo, los dos lenguajes de programación tienen sintaxis, semántica y usos muy diferentes.

Historia y Versiones

JavaScript fue inventado por Brendan Eich in 1995, para mejorar la experiencia de navegación y velocidad de las páginas web dentro del navegador Netscape. Se convirtió en standard ECMA en1997. El nombre oficial de este programa es ECMAScript y ha tenido diferentes versiones durante los años, los nombres de estas versiones se han abreviado de la siguiente forma ES1, ES2, ES3, ES5, y ES6.

Desde 2016 las nuevas versiones llevan el nombre del año (ECMAScript 2016 / 2017 / 2018).

ECMAScript 2009 o ES5, fue la primera revisión relevante de JavaScript. Entre las adiciones más significativas están el modo “strict mode”, soporte para el formato JSON con JSON.parse() y JSON.stringify(), String.trim() que quita el espacio en blanco a ambos lados de una cadena,Array.isArray() devuelve si un objeto es un Array, y métodos de iteración de Array como forEach().

ECMAScript 2015, ES6 o ECMAScript 6 fue la segunda revisión más importante de JavaScript. En esta versión se introdujeron las variables let y const, las Funciones Flecha, Array.find(), y Array.findIndex().

Microsoft finalizó todo el soporte para Internet Explorer en junio 15 de 2022 siendo este el final de este navegador. Internet Explorer no admite ES6 ni las últimas adiciones de JavaScript. El resto de navegadores si las soporta.

Sintaxis y Semántica

  • Distingue entre mayúsculas y minúsculas y utiliza el conjunto de caracteres Unicode.
  • No es necesario un punto y coma después de una declaración si está escrita en su propia línea. Pero si hay más de una declaración en una línea, entonces se deben separar con punto y coma. Aunque es considerado buena práctica hacerlo en todos los casos.
  • Los espacios, tabulaciones y caracteres de nueva línea son considerados como espacios en blanco por el intérprete e JS.
  • Comentarios: Según MDN los comentarios se comportan como espacios en blanco y se descartan durante la ejecución del script. Para realizar un comentario en una línea utilizamos //comentario y para un comentario de varias líneas /*comenrario*/ como lo hacemos en CSS.

Ejemplo:

/ un comentario de una línea

/* este es un comentario
* más largo, de varias líneas
*/

/* Sin embargo, no se pueden /* anidar comentarios */ se produce un SyntaxError */

Variables (var) (let) (const)

No es necesario definir el tipo de las variables: Una misma variable puede almacenar diferentes tipos de datos. JavaScript tiene tres tipos de declaraciones de variables. Deben comenzar con una letra, un guion bajo o un signo de dólar. Los siguientes caracteres también pueden contener dígitos pero no el primer carácter. Por ejemplo Numero5, _temp99, $credit y first-name.

Tipos

  • Var: Declara una variable con ámbito local o global, opcionalmente la inicia a un valor. Ahora su uso está recomendado sólo para casos muy específicos.
  • Let: Declara una variable local con ámbito de bloque, opcionalmente la inicia a un valor.
  • Const: Es más bien una constante. Declara un nombre de constante local de solo lectura, no se puede modificar su valor y tiene ámbito de bloque.

Ejemplos

//Declarar una variable:

let ejemplo
var ejemplo2
const ejemplo3

//Inicializar una variable:

let ejemplo = 5
var ejemplo2 = 5
const ejemplo3 = 5

Tipos de datos que se almacenan en una variable y la forma de expresarlos:

Tipos primitivos:

  1. Booleano: true y false.
  2. Null: Una palabra clave especial que denota un valor nulo o variable vacía.
  3. Undefined: Una propiedad cuyo valor no está definido.
  4. Number Un número entero o un número con coma flotante o decimal. Por ejemplo: 15 o 3.14159. No existe una diferencia entre un número entero y otro decimal, por lo que 1 y 1.0 son el mismo valor.
  5. String: O cadena de texto, es una secuencia de caracteres que representan un valor de texto. Cada carácter de la cadena se encuentra en una posición a la que se puede acceder individualmente, siendo el primer carácter el de la posición 0.Por ejemplo: "Hola".

Operadores

Asignación (=)

Este operador es el más utilizado y el más sencillo. Se utiliza para guardar un valor específico en una variable. El símbolo utilizado es =. A la izquierda del operador, debe indicarse siempre el nombre de la variable y al lado derecho del operador, se pueden indicar variables, valores, condiciones lógicas, etc:

Ejemplo

var numero1 = 3;

Operadores lógicos (&&) (||) (!)

AND es una operación lógica que obtiene su resultado combinando dos valores booleanos. El operador se indica mediante el símbolo && y su resultado solamente es true si los dos operandos son true:

variable1 variable2 variable1 && variable2
true true true
true false false
false true false
false false false

OR también combina dos valores booleanos, se indica mediante el símbolo || y su resultado es true si alguno de los dos operandos es true:

variable1 variable2 variable1 || variable2
true true true
true false true
false true true
false false false

Uno de los operadores lógicos más utilizados es el de la negación NOT. Se utiliza para obtener el valor contrario al valor de la variable, se indica al añadir el símbolo ! al inicio del identificador de la variable.

Ejemplo:

var visible = true;
console.log(!visible); // Muestra "false" y no "true"

variable1 !variable
true false
false true

Operadores de Comparación o relacionales (<) (>) (>=) (<=) (!=) (==)

Los operadores relacionales definidos por JavaScript son idénticos a los de matemáticas: mayor que >, menor que <, mayor o igual >=, menor o igual <=, distinto de != y el de igual que == que se utiliza para comparar el valor de dos variables (se debe diferenciar del operador de asignación =)

El resultado de todos estos operadores siempre es un valor booleano:

Ejemplo:

var numero1 = 3;
var numero2 = 5;
numero1 > numero2; // devuelve false
numero1 < numero2; // devuelve true

numero1 = 5;
numero2 = 5;
numero1 >= numero2; // devuelve true
numero1 <= numero2; // devuelve true
numero1 == numero2; // devuelve true
numero1 != numero2; // devuelve false

Operadores aritméticos (+) (-) (*) (/) (%)

JavaScript permite realizar manipulaciones matemáticas con el valor de las variables numéricas. Los operadores definidos son: suma +, resta -, multiplicación *, división / y módulo %.

Ejemplo:

var numero1 = 10;
var numero2 = 5;

numero1 / numero2; // = 2
3 + numero1; // = 13
numero2 - 4; // = 1
numero1 * numero 2; // = 50
numero1 % numero2; // = 0

Los operadores de incremento y decremento (++) (--)

El operador de incremento y decremento se indica mediante el símbolo ++ o -- en el nombre de la variable. Cuando los usamos como prefijo, su valor cambia antes de realizar cualquier otra operación. Si los utilizamos como sufijo, su valor cambia después de ejecutar la sentencia en la que aparece.

Ejemplo:

var numero = 5;
++numero;
//lo que equivaldría a
var numero = 5;
numero = numero + 1;

var numero = 5;
--numero;
//El anterior ejemplo es equivalente a:
var numero = 5;
numero = numero - 1;


var numero1 = 5;
var numero2 = 2;
numero3 = numero1++ + numero2;
// resultado: numero3 = 7 y numero1 = 6

var numero1 = 5;
var numero2 = 2;
numero3 = ++numero1 + numero2;
// resultado: numero3 = 8, se hace la adición antes de la operación (numero1 = 6)

El operador de asignación abreviada (+=)

Los operadores matemáticos también se pueden combinar con el operador de asignación para abreviar su notación:

Ejemplo:

var numero1 = 5;
numero1 += 3; // numero1 = numero1 + 3 = 8
numero1 -= 1; // numero1 = numero1 - 1 = 4
numero1 *= 2; // numero1 = numero1 * 2 = 10
numero1 /= 5; // numero1 = numero1 / 5 = 1
numero1 %= 4; // numero1 = numero1 % 4 = 1

Operador de concatenación (+)

La concatenación es cuando unimos dos o más cadenas, dando lugar a una nueva cadena. Para esto usamos el operador + de concatenación. Es también el símbolo que usamos para sumar números, aunque aplicado a las cadenas las unirá, se deben incluir los espacios para al final la frase no tenga palabras unidas.

Ejemplo:

const resultado = "esta noche " + "cenamos pizza";
resultado = “esta noche cenamos pizza”.

Operador condicional (?)

Es el único operador de JavaScript que toma tres operandos. El operador puede tener uno de dos valores según una condición. La sintaxis es:
condición ? val1 : val2
Si la condición es true, el operador tiene el valor de val1. De lo contrario, tiene el valor de val2.

Ejemplo:

var status = (age >= 18) ? 'adult' : 'minor';

Esta declaración asigna el valor "adult" a la variable status si age es de dieciocho años o más. De lo contrario, asigna el valor "minor" a status.

El operador coma (,)

Evalúa dos operandos y devuelve el valor del último operando. Este operador se utiliza principalmente dentro de un bucle for, para permitir que se actualicen múltiples variables cada vez a través del bucle.

Estructuras de Control de Flujo

Estructura If, If…Else

Esta estructura es usada para tomar decisiones en función de una condición.

if(condición) {
    ...
}

Si la condición se cumple, es decir, si su valor es true, se ejecutan las instrucciones que se encuentran dentro del bloque {...}. Si la condición no se cumple, o lo que es lo mismo, si su valor es false, no se ejecutan estas instrucciones del bloque y el programa continúa ejecutando el resto de instrucciones.

Ejemplo:

var mensaje = true;

if(mensaje) {
    console.log("Hola Mundo");
}

//también se puede expresar de la siguiente forma

if(mensaje == true) {
    console.log("Hola Mundo");
}

//o se puede asignar cualquier sentencia que al ser cumplida hará que se ejecute el código

var mensaje = true;
var mensaje2 = false;

if(!mensaje && mensaje) {
    console.log("Hola mundo");
}

Si queremos que ejecute un código cuando la condición sea falsa usaremos la palabra clave else.

if(condición) {
    ...
}

else {
     ...
}

Cuando la condición no se cumple, es decir, si su valor es false, se ejecutan todas las instrucciones contenidas en else { }

Ejemplo:

var edad = 18;

if(edad >= 18) {
     console.log("Adelante");
} else {
     console.log("Eres menor de edad");
}

En este caso, se mostraría el mensaje "Adelante".

La estructura if...else se puede encadenar para realizar varias comprobaciones seguidas:

Ejemplo:

if(edad < 12) {
    console.log("Eres un niño");
} else if(edad < 19) {
    console.log("Eres un adolescente");
} else if(edad < 35) {
    console.log("Eres un adulto joven");
} else {
    console.log("Eres mayor");
}

Estructura Switch

Esta estructura se utiliza cuando la condición que evaluamos puede tomar muchos valores.

Ejemplo:

switch(dia) {
case 1: console.log("Lunes"); break;
case 2: console.log("Martes"); break;
case 3: console.log("Miércoles"); break;
case 4: console.log("Jueves"); break;
case 5: console.log("Viernes"); break;
}

La cláusula puede ser una expresión al igual que en la estructura if. El comportamiento de la estructura switch es seguir evaluando cláusula por cláusula, aun cuando una de ellas haya cumplido la condición. Para evitar ese comportamiento, es necesario utilizar la sentencia break en las cláusulas que deseemos para que no se ejecute el resto.

Estructura For

Esta estructura permite realizar bucles. El bucle se repite mientras una condición se siga cumpliendo y después de cada repetición, actualiza el valor de las variables que se utilizan en la condición.
En la inicialización se establecen los valores iniciales de las variables que controlan la repetición, la condición es el único elemento que decide si continua o se detiene la repetición y la actualización es el nuevo valor que se asigna después de cada repetición a las variables que controlan la repetición.

Ejemplo:

Let mensaje = “buenos días”

for(let i = 0; i < 5; i++) {
    console.log(mensaje);
}

Los bucles se siguen ejecutando mientras se cumplan las condiciones y se dejan de ejecutar cuando la condición no se cumpla. En este ejemplo, mientras la variable i valga menos de 5 el bucle se ejecuta.
Normalmente, la variable que controla los bucles for se llama i. El siguiente código recorre el Array y muestra cada dia de la semana.

Ejemplo:

var dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

for(var i=0; i < 7; i++) {
     console.log(dias[i]);
}

Estructura While

La estructura while ejecuta un bucle simple, mientras se cumpla la condición.

Ejemplo:

var cuenta = 0;

while(cuenta < 7) {
    console.log("Mensaje " + cuenta);
    cuenta++;
}

Mientras la condición indicada se siga cumpliendo, repite la ejecución de las instrucciones definidas dentro del bloque. Es importante modificar los valores de las variables incluidas dentro de la condición, ya que otra manera, el bucle se repetiría infinitamente, bloqueando la ejecución del resto del script.

Estructura Do…While

La diferencia de esta estructura a la anterior es que el bloque del bucle se ejecuta la primera vez sin comprobar la condición. Después el funcionamiento es idéntico al while.

Ejemplo:

var cuenta = 0;

do { console.log("Mensaje " + cuenta);
     cuenta++;
}while(cuenta < 10);

Estructura For... In

Es una estructura de control derivada de for, implica el uso de objetos y sus propiedades, permitiendo recorrer las propiedades de un objeto. En cada iteración, un nuevo nombre de propiedad del objeto es asignado a la variable: Si se quiere recorrer todos los elementos que forman un array, la estructura for...in es la mejor forma de hacerlo.

Ejemplo:

var dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

for(i in dias) {
     console.log(dias[i]);
}

Funciones, objetos y Arrays

Funciones

Las funciones en JavaScript se crean mediante la palabra reservada function, seguida del nombre de la función. El nombre de la función junto con sus argumentos entre paréntesis se utilizan para llamar a esa función cuando sea necesario. Se les debe asignar un nombre único para poder utilizarlas dentro del código. Los símbolos { } se utilizan para contener todas las instrucciones que pertenecen a la función.

function nuevoCliente (lista de argumentos) {
    instrucciones
    return resultadoDeLasInstrucciones
}

Tembién existen las funciones flecha, en donde lo anterior se puede expresar de la siguiente forma:

let nuevoCliente = lista de argumentos => instrucciones

Objetos

Es una lista de varios elementos compuestos de un nombre y un valor. O un grupo de propiedades y funciones a los que se accede a través de una clave. Las funciones, arrays, expresiones regulares también son consideradas objetos.

Ejemplo:

{type:"Fiat", model:"500", color:"white"};

Arrays

Se trata de una lista de elementos que contenemos entre los símbolos [] y separamos con comas, en JS los arrays pueden contener varios tipos de expresiones. El primer valor recibe la propiedad de nombre o índice '0', la segunda de '1', y así sucesivamente.

[gato, perro, pez, 10, 5]
